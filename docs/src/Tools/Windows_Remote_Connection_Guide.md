# Windows远程链接指南
快速解决Windows远程链接问题

## 前提
这里我们只考虑内网情况下，也就是控制机与被控制机在同一网络下（最通俗易懂的就是同一个Wifi下）

1. 在同一网络下
2. 远程桌面功能已开启（打开电脑的设置，搜索：`远程桌面`，在其下把开关打开就行）
3. 系统是专业版

## 操作
1. 在被控制机上，打开cmd，输入`ipconfig`，找到自己的IPV4地址，记录
2. 在控制机上，打开远程桌面链接，计算机一栏输入步骤1记录的IPV4地址
3. 然后会弹出要凭证（没要凭证就是上面有问题，防火墙没开远程的端口/远程桌面的开关没开/远程桌面的服务没起来，按这几个关键词百度一下就行，理论上我电脑22年的，都是默认打开的，没有调过）
4. 凭证这里：
    * 用户名，在：`计算机 > 系统盘（一般是C盘） > 用户（或者叫Users）`下，找到自己用的那个用户名，输入就行
    * 凭证，你的微软账户登录密码（非常难崩）

通过以上步骤，成功建立了远程链接；

弱网下，使用感比要中转软件（XX远程、xxxflow等）的好很多；

## 公网访问
目前暂时用的还是`UU远程`，如果也想要通过上面步骤来实现远程链接，可以尝试用`Sakura映射`来内网穿透出去试试看，当然，警惕电脑被黑的风险！
